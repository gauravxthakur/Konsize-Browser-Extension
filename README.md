Since this extension is not published on a web store, you will need to **sideload** it into your Chromium-based browser (like Chrome, Brave, Edge, Opera, etc.).

### 1. Download the Files

1.  **Clone the Repository:** If you are familiar with Git, clone this repository to your local machine:
    ```bash
    git clone https://github.com/gauravxthakur/Konsize-Browser-Extension.git
    cd Konsize-Browser-Extension
    ```
2.  **Download as ZIP:** Alternatively, you can download all the files as a ZIP archive and extract them to a folder on your computer.

### 2. Load the Extension in Your Browser

This process involves enabling "Developer Mode" in your browser's extension settings.

1.  **Open the Extensions Page:**
    * In your browser, navigate to the extensions management page.
    * A common URL is `chrome://extensions` or `edge://extensions` (just replace `chrome` with your browser name). 

2.  **Enable Developer Mode:**
    * Look for a toggle switch labeled **"Developer mode"** in the top-right corner of the extensions page and click it to turn it **ON**.

3.  **Load the Extension:**
    * Three new buttons should appear. Click on the **"Load unpacked"** button.

4.  **Select the Folder:**
    * A file dialog will open. Navigate to and select the folder where you downloaded and extracted the `Konsize AI Summarizer` files.
